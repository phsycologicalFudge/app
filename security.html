Security at ColourSwift Cloud

We built ColourSwift Cloud with security as the foundation, not an afterthought. Below you’ll find a clear overview of how your data is protected across infrastructure, networks, storage, and accounts.

Infrastructure Protection

  • UK colocation facility with 24/7 monitoring, controlled access, redundant power, and cooling.
  • Physical rack security — only authorised engineers can access hardware.
  • Capacity planning ensures uptime during maintenance or hardware faults.

Network & Access Controls

  • Cloudflare shields public endpoints — real server IPs stay hidden.
  • Dedicated tunnel and reverse proxy isolate core services from the internet.
  • Firewall rules and fail2ban mitigate brute force and suspicious traffic.
  • Minimal open ports; admin access is tightly restricted and logged.

Data Encryption

  • TLS (HTTPS) enforced everywhere with modern ciphers and HSTS.
  • All server storage is encrypted — stolen disks remain unreadable.
  • Optional end-to-end encryption available in supported clients.
  • Encrypted backups stored separately and tested regularly.

Account Security

  • Two-factor authentication (2FA) available for all users; mandatory for admins.
  • Role-based access tied to your subscription plan and permissions.
  • Failed logins are monitored and automatically rate-limited.

Privacy & Compliance

  • UK-hosted and GDPR-aligned. Data stays in the UK unless you request otherwise.
  • No advertising, no mining — your files are your files.
  • Clear policies explain what data we collect, why, and how it’s used.

Our Promise

No provider can claim perfect security. Instead, we apply a layered defence model used by larger enterprises and continually improve it. If you require a formal statement of controls for compliance, we can provide one on request.
